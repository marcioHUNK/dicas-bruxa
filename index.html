  • Ataque corpo-a-corpo com uma espada curta. Arma fácil de manusear
  • Alcance: Corpo-a-corpo (1,5m)
  • Ataque: 1d20 + mod. destreza + Proficiencia
  • Dano: 1d6 + md. destreza (enquanto sua destreza for 13, seu mod destreza é +1)
  • Quando usar: Quando não puder ou quiser usar magias. Quando estiver quando precisar acertar um inimigo corpo-a-corpo
  • Caracteristica adquirida do pacto. Permite você influenciar a mente de outros seres para como eles te veem.
  • Alcance: A distância (3m)
  • Ativação: Selecionar os alvos. Os alvos devem resistir a um teste de sabedoria ou será afetados pela característica.
  • Dano: Essa caracteristica não dá dano, mas se o alvo falhar, você deve escolher se quer encantar ou amedrontar os alvos. Se ele passar, nada acontece.
  • Quando usar: Sempre que precisar influenciar a mente de outros seres. Você pode usar essa caracteristica uma vez por descanso curto
  • Magia inata adquirida do pacto. Permite você se teletransportar para um local dentro do alcance. É uma ação bônus.
  • Alcance: A distância (até 9m).
  • Ativação: selecione a magia. Esteja atento para selecionar a distancia de até 9m ou 30f se o mapa não estiver em metros. Clique onde deseja se reposicionar.
  • Dano: Essa caracteristica não dá dano
  • Quando usar: Quando precisar se reposicionar, se teleportar para um lugar seguro ou dentro de algum lugar. pode ser usada uma vez por descanso longo
